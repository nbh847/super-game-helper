# 标注策略指南

## 不同类型对局的标注策略

### 总体原则

根据[对局筛选指南](game_selection_guide.md)中的分类，采用不同的标注策略。

### 胜局标注

#### 标注策略
- **标注间隔**：每15秒1帧
- **覆盖范围**：完整对局（从出门到结束）
- **用时**：20-30分钟/局（假设20分钟对局，约80-120帧）

#### 重点关注
1. **补刀阶段**（0-8分钟）
   - 小兵最后击杀时机
   - 对线补刀节奏
2. **战斗阶段**（8-15分钟）
   - 技能释放时机
   - 连招配合
3. **运营阶段**（15-20分钟）
   - 团战参与
   - 资源分配

#### 数据价值
- ⭐⭐⭐⭐⭐⭐ 最高
- 学习"正确的"战斗和运营策略
- 包含完整的游戏流程

### 高KDA败局标注（KDA ≥ 1.0）

#### 标注策略
- **标注间隔**：每15秒1帧
- **覆盖范围**：完整对局
- **用时**：20-30分钟/局

#### 重点关注
1. **输出效率**
   - 伤害/死亡比高
2. **位置选择**
   - 团战站位
   - 追击时机
3. **资源管理**
   - 经济利用效率
   - 装备选择时机

#### 数据价值
- ⭐⭐⭐⭐ 高价值
- 学习"有输出但如何避免被翻盘"
- 失败原因分析：是走位失误？技能空放？还是经济落后？
- **DQN训练作用**：
  - 死亡惩罚（-5.0）会让AI学习避免这些导致死亡的行为
  - 高KDA败局中的"错误行为"会被严重惩罚
  - AI自然学会避免这些模式

### 中KDA败局标注（0.5 ≤ KDA < 1.0）

#### 标注策略
- **标注间隔**：每15秒1帧
- **覆盖范围**：完整对局
- **用时**：20-30分钟/局

#### 重点关注
1. **正常对线**
   - 补刀、换血、技能消耗
2. **参与团战**
   - 适时支援
   - 后期运营

#### 数据价值
- ⭐⭐⭐ 中价值
- 学习基本的运营和战斗意识
- 输出中等，但有一定作用

### 前期标注（低KDA败局）

#### 标注策略
- **标注间隔**：每15秒1帧（仅前8-15分钟）
- **覆盖范围**：仅前60-80%时间
- **跳过后**：后期崩盘阶段
- **用时**：10-15分钟/局

#### 重点关注
1. **出门阶段**
   - 出装顺序
   - 买装备时机
2. **对线阶段**（前3-5分钟）
   - 第一波兵线处理
   - 第一次交战

#### 数据价值
- ⭐⭐⭐ 中等价值
- 最大化有用学习时间
- 避免负面经验污染
- 适合：被单杀、被压制、但前期正常发挥的对局

## 标注时间分配建议

### 每周工作量

假设每周10局，胜率44%（约4.5胜 + 5.5败）：

| 对局类型 | 局数 | 标注间隔 | 标注用时 |
|---------|------|---------|---------|
| 胜局 | 4.5 | 15秒/帧 | 1.5-2.3小时 |
| 高KDA败局 | 约1.5 | 15秒/帧 | 0.5小时 |
| 中KDA败局 | 约1.5 | 15秒/帧 | 0.5小时 |
| 低KDA败局（前8分钟标注） | 约1 | 15秒/帧，后期跳过 | 10-15分钟浏览 + 10分钟标注 = 20-25分钟 |
| **总计** | **8.5局** | - | **2.75-3.8小时** |

### 时间优化建议

1. **批量处理**：一次性加载所有视频，批量标注
2. **快捷键熟练**：标注工具快捷键操作提速
3. **快速判断**：KDA客观判断30秒内完成
4. **跳过低价值**：低KDA败局快速浏览后立即决定跳过

## DQN训练与数据价值

### 为什么败局值得训练？

**高KDA败局的价值**：
- DQN的奖励设计：
  - 补刀奖励：+1.2（最高权重）
  - 击杀奖励：+0.8
  - 死亡惩罚：-5.0（严重惩罚）

**训练效果**：
- 高KDA败局中的死亡会被严重惩罚（-5.0）
- AI自然学习避免导致死亡的行为
- 相当于给AI提供了"反面教材"
- 不需要手动删除败局，DQN会自动处理负样本

### 混合训练策略

**所有对局混合训练**：
- 胜局：学习"正确的"战斗和运营
- 高KDA败局：学习"有输出但避免被翻盘"
- 中KDA败局：学习基本的运营意识
- 低KDA败局：跳过或仅标注前期

**预期效果**：
- ✅ AI学会完整的游戏流程
- ✅ 避免负面模式污染
- ✅ 提升泛化能力
- ✅ 比只选胜局训练更全面
