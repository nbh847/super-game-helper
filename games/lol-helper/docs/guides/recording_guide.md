# 游戏录制指南

## 游戏内置录制功能说明

### 录制功能位置
- **回放界面**：红色录制按钮（左上角）
- **默认格式**：MP4
- **默认分辨率**：720p
- **保存位置**：可在客户端设置中自定义

### 推荐录制设置
- **倍速**：2倍速
  - 理由：节省一半录制时间（20分钟 → 10分钟）
  - 不影响录制视频的质量
- **分辨率**：1080p（最佳训练质量）
  - 可在游戏设置中调整
- **编码**：H.264
  - 平衡质量和文件大小

## 录制流程

### 标准流程
1. 打开英雄联盟客户端
2. 进入"战绩"页面
3. 找到要录制的对局
4. 点击"观看回放"按钮
5. 等待回放加载
6. 设置回放速度为2倍速
7. 点击红色录制按钮开始录制
8. 等待回放完成
9. 录制视频自动保存到指定目录

### 快速录制技巧
- **批量录制**：可以提前进入多个回放，依次录制
- **后台录制**：可以一边录制一边做其他事情
- **存储检查**：定期检查保存位置，确保视频保存成功

## 文件命名规范

### 推荐格式
```
{英雄名}_{局数}_{日期}.mp4

示例：
- Lux_05_20240122.mp4
- Jinx_03_20240123.mp4
- Teemo_12_20240124.mp4
```

## 常见问题

### Q: 录制后找不到视频在哪里？
A: 打开英雄联盟客户端 → 设置 → Replays → 查看保存位置

### Q: 录制的视频卡顿怎么办？
A:
1. 检查游戏是否后台运行太多程序
2. 降低录制分辨率到720p
3. 关闭不必要的程序

### Q: 2倍速录制会影响质量吗？
A: 不会。2倍速回放只是加快浏览速度，录制的是游戏实际渲染的画面，质量不受影响。

## 录制检查清单

每局录制前确认：
- [ ] 视频保存位置已设置
- [ ] 回放速度设置为2倍速
- [ ] 分辨率设置为1080p（可选）
- [ ] 硬盘空间充足（每局约500MB-1GB）
